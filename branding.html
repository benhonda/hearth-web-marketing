Ingredients · Serves 4
  • 1 whole chicken, about 4 lb
  • 1 lemon, halved
  • 6 sprigs fresh thyme
  • 4 cloves garlic, smashed

Method

04 / Typography

The type system

Five tiers. Marcellus is reserved for the wordmark. Lora carries display headings, italic, and editorial moments. DM Sans carries small-caps labels and body copy. The hand-offs between serifs and sans are what give Hearth its publishing-house feel without slipping into old-fashioned.

01 · Display (wordmark only)
HEARTH
Marcellus · single weight (400)
Reserved for the wordmark on spines, covers, app icon, and marketing — nowhere else. No italic (there isn’t one). Always gold foil on sage or ochre on cream.
02 · Heading / H1

Lora · regular (400)
Recipe titles, chapter titles, section openers. Warm transitional serif that reads editorial without feeling old. Never bold — hierarchy from size, not weight.

Lora italic
Ledes, metadata, italic Roman-numeral step markers, folios. Carries every italic job — Marcellus never does, because it has no true italic.

06 / Decisions pending

Pick from the options

Six remaining decisions to finalize the brand book. Each subsection shows the options side-by-side. Pick one per subsection and I’ll lock it in.

6a

Shape & radius Locked

Pill buttons · square (2px) inputs · 2px cards. Buttons feel warm/inviting; inputs stay tight and editorial; cards read like book pages.

HEARTH

6b

Motion Locked

Restrained. Every state change fades in ~280ms. No bouncing, no springs, no overshoot. Motion is soft and unobtrusive — users shouldn’t notice it, that’s the point. No "craft moments" layer (it would risk feeling gimmicky).

Save recipe
Auto-loops between “Save recipe” and “Saved ·” to show the fade. Apply this 280ms ease transition to all interactive state changes — hover, focus, click, page navigation.
6c

Photography Locked

Photography lives in two places: recipe-detail hero (full-bleed at the top of each recipe) and library thumbnails (small square on each list row). The rest of the brand — book covers, chapter openers, cookbook interiors — stays typographic. User-uploaded photos, no AI enhancement.

Voice & tone Locked

Personal but restrained. Warmth from phrasing (your library, tucked away), never from emoji or exclamation marks. Factual when factual serves — “50 min · Serves 4” stays that way. Conversational when it helps — empty states, confirmations, conversion.

6e

Button system Locked

Six button tokens total: one primary, two secondaries, two tertiaries, one inline link. All pill-shaped except tertiary text / inline links (no container). Each has a clear usage rule so code stays consistent.

Primary
When: the single most important action on a screen — save, make a cookbook, place order. Never more than one primary per view.
Style: Foil Light #D4B062 · Warm Charcoal text · pill · soft shadow.
Secondary · two styles
S1 · Outlined — "alternate high-signal action"
When: a secondary action that should still be clearly tappable — e.g., next to a Primary (“Save” / “View”). Book-jacket feel; editorial.
S3 · Soft filled — "quiet supporting action"
When: background / supporting actions — pagination, filter toggles, “Show more,” chip-like tags. Quieter than S1.
Tertiary · two styles
T1 · Plain text — "low-stakes or destructive"
When: Cancel, Skip, Delete, Close — actions that shouldn’t compete for attention. Hover reveals underline.
T2 · Text + chevron — "forward navigation"
When: going deeper — “See all,” “View original,” “Read the note.” Chevron nudges right on hover (restrained).
Inline link · one style

Originally from Food52’s roast chicken guide. Edited for a smaller bird, with lemon and thyme swapped for rosemary.

When: any inline link inside body prose. Underline is always visible (like a published book), in the same ink color as the body — never gold. Hover dims slightly.
Rule of thumb
One Primary per view. Pair with S1 (outlined) when there’s an equal-weight alternative. Use S3 (soft filled) for rows of quiet actions. Drop to T1 (plain text) for Cancel/Delete. T2 (text + chevron) is reserved for forward-navigation patterns only. Inline prose gets L2 links.
